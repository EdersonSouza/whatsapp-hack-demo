# Demonstração Educacional - Golpe do WhatsApp

Esta é uma demonstração educacional que mostra como hackers podem capturar códigos de verificação do WhatsApp.

## URLs:
- **Página da Vítima**: `https://[seu-app].onrender.com/vitima`
- **Painel do Hacker**: `https://[seu-app].onrender.com/hacker`

## Uso Educacional:
1. Abra o Painel do Hacker no projetor
2. Envie o link da Vítima para a plateia
3. Peça para digitarem códigos de 6 dígitos
4. Mostre em tempo real como aparecem no painel
